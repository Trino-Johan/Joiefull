# 📱 Joiefull - Consultation d'articles de mode

**Application iOS pour consulter, commenter et aimer des articles de mode.**

[![Swift](https://img.shields.io/badge/Swift-5.9%2B-orange.svg?style=flat&logo=swift)](https://swift.org)
[![SwiftUI](https://img.shields.io/badge/SwiftUI-5.0%2B-blue.svg)](https://developer.apple.com/xcode/swiftui/)
[![iOS](https://img.shields.io/badge/iOS-17.0%2B-lightgrey.svg)](https://developer.apple.com/ios/)

## ✨ Fonctionnalités principales

- ✅ Consulter la description des articles
- ✅ Catalogue d'articles de mode
- ✅ Liker les différents articles
- ✅ Commenter les articles

## 🏗️ Stack technique

- **Architecture** : MVVM
- **Données** : API "https://github.com/OpenClassrooms-Student-Center/Cr-ez-une-interface-dynamique-et-accessible-avec-SwiftUI.git"
- **UI/UX** : SwiftUI natif
- **Outils** : Xcode 26.0.1, iOS 17+
